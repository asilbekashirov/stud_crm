[Mega Dream](https://stud_crm.vercel.app/) is a website for organization that helps students to apply for universities abroad.

## Basic Info:

The website has a list of available universities to apply for. Students choose the universities they like, upload their documents and the rest of the work is handled by the organization.

### Tech Stack:

- NextJs
- React
- TypeScript
- TailwindCSS
- Redux, Redux toolkit, RTKQuery
- React Hook Form
- Framer Motion
